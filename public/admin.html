Lots & Timers

Set a duration (seconds) and press Start to open bidding. The countdown is enforced by the server — bids after it ends are rejected.

Clear bids only wipes every bid and resets each lot to draft (timer/winner reset), keeping the lots themselves. Clean up & start a new auction removes the lots too. Users and Terms & FAQ are kept either way.

Download credentials (User ID + Password)

Stored passwords are encrypted (bcrypt) and cannot be read back. These buttons therefore regenerate fresh temporary passwords for the chosen users and download them as a CSV (Name, User ID (email), Password, Role). Existing passwords for those users stop working — distribute the new ones. Logged-in sessions stay active until logout.

Bulk upload from Excel

Upload an .xlsx or .csv with a header row containing Name and Email columns (an optional Role column may be bidder or admin). A temporary password is generated for each new user — copy the results below and share them securely. Download a template.
