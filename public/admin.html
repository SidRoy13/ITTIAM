Download credentials (User ID + Password)

Stored passwords are encrypted (bcrypt) and cannot be read back. These buttons therefore regenerate fresh temporary passwords for the chosen users and download them as a CSV (Name, User ID (email), Password, Role). Existing passwords for those users stop working — distribute the new ones. Logged-in sessions stay active until logout.
